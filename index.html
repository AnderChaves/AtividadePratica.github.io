Portfolio

Análise de Dados com Python

Desenvolvi um projeto de análise de dados utilizando Python, onde realizei a limpeza, análise e visualização de dados de um grande conjunto de dados de vendas. Utilizei as seguintes bibliotecas:

  • Pandas: Para manipulação e análise de dados.
  • NumPy: Para operações numéricas e manipulação de arrays.
  • Matplotlib: Para criação de gráficos e visualização de dados.
  • Seaborn: Para visualizações estatísticas aprimoradas.

O projeto envolveu a análise de tendências de vendas, identificação de padrões sazonais e geração de relatórios detalhados para suporte à tomada de decisões.
